## 准备工作

* nodejs
* git
* hexo安装 (npm install -g hexo)
* hexo-deployer-git(用git部署需要这个包 npm install --save hexo-deployer-git)

## 使用

安装好 hexo 包后，打开 git bash 命令行窗口与hexo交互。

~~~
# 初始化一个博客程序
hexo init

# 查看 hexo 的帮助
hexo -h

# 生成静态文件
hexo generate
~~~
